Process

Initially, I wanted to make a covid tracker that would take a Country's Alpha 3 code and a date as input and then return information. I later realised that it would require effort for the user to find the country alpha 3 code so i chose to add the restcountries API


Now the page first searches for the country data using the RestCountries and then displays that. It then uses the country's alpha 3 code which is returned from the RestCountries object and the date(s) provided by the user and looks for thr covid data and policy data for the country.


As soon as the RestCountries API finds the country data it sets the Name and Flag variables in the local storage to get the next time the user returns.
